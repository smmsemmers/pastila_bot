1) НАСТРОЙКА В BOTFATHER
--------------------------------------------
Открой @BotFather → /mybots → выбери бота → Edit Bot. Для каждого:
• Edit About — вставь текст «About» (см. ниже)
• Edit Description — вставь текст «Description»
• Edit Botpic — загрузи логотип
• Bot Settings → Allow Groups? → Enable (чтобы можно было добавлять в чаты)
• Bot Settings → Group Privacy:
    - у @PastilaTaskBot и у бриджа → OFF
    - у GPT-бота → можно не трогать
Команды по «/» руками НЕ вписывай — они ставятся автоматически.

3) АППРУВ ГРУПП (новое)
--------------------------------------------
Когда кого-то добавляют бота в новую группу — тебе в личку приходит запрос
с кнопками ✅ Разрешить / ❌ Запретить. До одобрения бот в группе молчит.
✅ → бот работает и запоминает группу. ❌ → выходит из группы.
Одобряешь пока только ты. Текущая рабочая группа уже разрешена.

На Render (у обоих воркеров) должны стоять переменные:
  ADMIN_USER_IDS = 41082373   (твой Telegram ID)
  GROUP_APPROVAL = true
Чтобы добавить Глеба вторым аппрувером — впиши его ID через запятую:
  ADMIN_USER_IDS = 41082373,<ID_Глеба>
(ID можно узнать: пусть Глеб напишет любому боту /id)


4) ЗАЧЕМ ТРИ ОТДЕЛЬНЫХ БОТА
--------------------------------------------
🤖 TaskBot — «секретарь»: всегда онлайн, дёшево, задачи и напоминания.
💬 GPT-бот — «эрудит с интернетом»: включается для поиска и вопросов.
💻 Бридж — «инженер»: живёт на Маке, умеет код/деплой/файлы.
Одним ботом не собрать: у каждого своё «место работы» и свои задачи.
Чтобы не перебивали друг друга — при живом бридже TaskBot молчит на
свободный текст (флаг BRIDGE_PRIMARY), первым отвечает бридж.
