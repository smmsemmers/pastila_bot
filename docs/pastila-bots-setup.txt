1) НАСТРОЙКА В BOTFATHER
--------------------------------------------
Открой @BotFather → /mybots → выбери бота → Edit Bot. Для каждого:
• Edit About — короткий слоган (лимит 120 символов)
• Edit Description — подробнее (лимит 512): вот это я + ключевая задача + функции
• Edit Botpic — загрузи иконку (icon-*.png)
• Bot Settings → Allow Groups? → Enable
• Bot Settings → Group Privacy: у Pastila Code и TaskBot → OFF; у GPT — можно не трогать
Команды по «/» руками НЕ вписывай — ставятся автоматически.
