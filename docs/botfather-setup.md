# 🍬 Настройка ботов Pastila OS в BotFather

Гайд для трёх ботов: **@PastilaTaskBot**, **@pastila_gPT_remote_bot**, **@pastila_code_remote_bot** (бридж).
Всё делается в [@BotFather](https://t.me/BotFather) — никакого кода.

---

## 🚀 Быстрый чек-лист

Для **каждого** бота пройди по шагам:

- [ ] `/mybots` → выбрать бота → **Edit Bot**
- [ ] **Edit About** → вставить строку About (профиль, до 120 симв)
- [ ] **Edit Description** → вставить Description (пустой экран чата, до 512 симв)
- [ ] **Edit Botpic** → загрузить аватар (лого пастилы)
- [ ] **Bot Settings → Group Privacy** → выставить по таблице ниже
- [ ] **Bot Settings → Allow Groups?** → **Enabled**

> ⚠️ **Команды (`Edit Commands`) заполнять НЕ нужно** у TaskBot и GPT-бота —
> они регистрируются автоматически при запуске (`setMyCommands` в коде).
> У бриджа слэш-команд нет.

---

## 🔧 Что где заполнять

| Пункт в BotFather | Что это | Лимит |
|---|---|---|
| **Edit Name** | Имя бота (видно везде) | — |
| **Edit About** | Короткая строка в карточке профиля | 120 символов |
| **Edit Description** | Текст на пустом экране чата «What can this bot do?» | 512 символов |
| **Edit Botpic** | Аватар бота | картинка |
| **Edit Description Picture** | Картинка/видео над описанием (необязательно) | — |
| **Edit Commands** | Меню команд по «/» — *задаётся кодом, руками не трогать* | — |

### 🔒 Group Privacy (Bot Settings)

| Бот | Group Privacy | Почему |
|---|---|---|
| 🤖 **@PastilaTaskBot** | **OFF** | иначе не видит обычные сообщения и файлы в группе |
| 💻 **@pastila_code_remote_bot** (бридж) | **OFF** | главный ассистент — должен видеть весь текст и файлы |
| 💬 **@pastila_gPT_remote_bot** | ON (можно) | реагирует только на команды, упоминания и reply |

**Allow Groups → Enabled** у всех трёх.

## 🧩 Зачем три отдельных бота, а не один

Каждый — под свою роль, чтобы не мешать друг другу:

| Бот | Роль | Где живёт | Суть |
|---|---|---|---|
| 🤖 **TaskBot** | «Секретарь» | сервер 24/7 | надёжный и дешёвый: задачи, таблица, напоминания. Тяжёлый ИИ ему не нужен. |
| 💬 **GPT-бот** | «Эрудит с интернетом» | по требованию | другой движок: веб-поиск, OCR, отдельная оплата токенов. |
| 💻 **Бридж** | «Инженер» | твой Мак | умеет то, что серверу нельзя: файлы, код, деплой, экспорты. |

**Почему нельзя одним ботом:**
один должен спать на сервере и быть дешёвым, другой — лазить в интернет,
третий — иметь доступ к железу. Разделили по «мозгам» и по месту работы.

**Чтобы не перебивали друг друга:** когда бридж включён, TaskBot молчит на
свободный текст, голос и файлы (флаг `BRIDGE_PRIMARY` / `TASKBOT_COMMAND_ONLY`) —
первым отвечает бридж, а TaskBot остаётся на свои команды и напоминания.
